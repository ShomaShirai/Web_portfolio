# Webオセロゲーム作成
## ポートフォリオ用

## 仕様
- 8×8のオセロボードを実装
- 現在のプレイヤーや得点を表示
- パス，リセット，自動モードを利用可能

## 開発環境
- Visual studio code
- HTML
- CSS
- javascript

## 工夫した点
- プレイヤーのターンになった場合に駒が置ける位置を強調している
- 駒を置く場合にhoverしている
